# WEBPACK


Для установки пактів пропишіть npm insltall

## Команди

### Запуск сервера
```shell
npm run start
```

### Збірка для оптимізації
```shell
npm run build-dev
```

### Збірка з оптимізацією
```shell
npm run build-prod
```

### Очищення dist
```shell
npm run clear
```
